EXECUTION DES INSTRUCIONS DE 1 à 13

  • Créez un dossier appelé learn_git.
  • Cd (changer de répertoire) dans le dossier learn_git.
  • Créez un fichier appelé third.txt.
  • Initialisez un dépôt git vide.
  • Ajoutez third.txt à la zone de transit.
  • Validez avec le message « ajout de third.txt ».
  • Vérifiez votre commit avec git log.
  • Créez un autre fichier appelé fourth.txt.
  • Ajoutez fourth.txt à la zone de stockage.
  • Validez avec le message « ajout de fourth.txt »
  • Supprimez le fichier third.txt.
  • Ajoutez cette modification à la zone de transit. (En utilisant la commande « git add .
  • Validez avec le message « suppression de third.txt »

EXECUTION DES INSTRUCIONS DE 13 à 16

  • Validez avec le message « suppression de third.txt »
  • Vérifiez vos commits à l’aide de git log.
  • Modifiez vos paramètres globaux en core.pager=cat - vous pouvez en savoir plus à ce sujet ici.
  • Écrivez la commande appropriée pour répertorier toutes les configurations globales de git sur votre machine.

EXECUTION DES INSTRUCIONS DE 16 à 17

  • Écrivez la commande appropriée pour répertorier toutes les configurations globales de git sur votre machine.
